3) Troubleshooting View
==================================================

Issues solved
- 405 on GET user endpoint traced to missing GET mapping and fixed.
- bootRun failure traced to missing main datasource config.
- TypeScript "--jsx not set" and module declaration issues fixed via tsconfig/dependency updates.
- Expo default startup screen issue fixed by root-route configuration.
- npm ci lock mismatch fixed by manifest and lockfile synchronization.
- Sonar warnings fixed for hardcoded secrets and test quality issues.

Important lessons learned
- Postman path vars must resolve to real values, not literal placeholders.
- Local Windows EPERM unlink errors during npm ci can be caused by locked native files.
- Stale IDE diagnostics can remain after fixes and may require language-server/project reload.
